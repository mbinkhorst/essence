    HOW TO SELECT A PROGRAM (HIGH, MEDIUM OR LOW)
    OBSERVE make sure that the Bun Warmer is in the STANDBY ON mode.
    PRESS the MODE button
     
    one time to select the LOW program. The LOW indicator
     
    of the button flashes until temperature is reached. Then illuminates.
    PRESS the MODE button
     
    again to select the MEDIUM program. The MEDIUM indicator
     
    of the button flashes until temperature is reached. Then illuminates.
    PRESS the MODE button
     
    again to select the HIGH program. The HIGH indicator
     
    of the button flashes until temperature is reached. Then illuminates.
