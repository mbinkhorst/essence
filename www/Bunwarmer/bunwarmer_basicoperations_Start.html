    OPEN DOOR DETECTION
    How to START a program
    OBSERVE make sure that the door is closed and the primary latch and the secondary latch are locked correctly.
    PRESS the ON/OFF button once. The light ring of the
     
    button turns to the color Green to indicate that the Bun Warmer is in the STANDBY ON mode.
    All three indicators
     
    of the MODE button flash which indicates that the default HIGH program is started. The MODE indicator will illuminate when
     
    the unit has reached its temperature.
    After 60 minutes the program stops automatically. The unit goes to the STANDBY OFF mode. The indicator
     
    of the MODE button goes OFF and the light ring of the
     
    button turns to the color Blue.
