ESSENCE® CONVECTION OVEN
    quick reference guide
    FROZEN MEALS
    In case of deep frozen meals, use the Frozen Meals program before starting a desired program.
    ERROR RESET
    In case of an unexpected error message, press and hold the
     
    button for 6 seconds to reset the oven.
    OVEN CONTENT
    1. Using foil/wax paper, is only allowed to use when it is heat resistant. Using material with low heat resistance can cause overheat or burning smell situations.
    2. To obtain the best performance, do not overload the oven. The oven is designed for a maximum load of 32 meals.
    CLEANING
    Make sure that the oven cavity is checked and if necessary cleaned after each cooking program.
    PAUSE FUNCTION
    If needed to open the door during a cooking program, press the
     
    button before opening the door.
